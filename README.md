# ninja-graphql-react-mongo
- Created with React, Mongodb, Graphql

# Installation
`npm install`

# Dev at localhost
`npm run dev`

# Build
`cd client/ && npm run build`

# Server only
`npm run server`

# Client only
`npm run client`
